# GoodPlugins

**QQChannelChatGPT插件**

支持功能：
- 以图搜番（https://trace.moe）
- 随机动漫图片（https://t.mwm.moe/）
- 喜报

安装方法：
```
plugin i https://github.com/Soulter/goodplugins
```

使用方法：
- `/搜番`**并且**加上图片即可搜番
- `/moe`即可调用随机动漫图片
- `/喜报 <你的内容>`
- `/mcs`: `/mcs <minecraft服务器地址>`即可查询服务器状态
- `/一言`即可调用一言
- `/今天吃什么`：随机选择吃什么
  - `/今天吃什么 添加 美食1 美食2 ...`：添加美食
  - `/今天吃什么 删除 美食1 美食2 ...`：删除美食
- `/喜加一`：EPIC 喜加一
- `早安/晚安`：在群里发早晚安，记录睡眠时长，保持健康！
